# Company Fullstack Project

이 저장소는 회사 웹사이트의 프론트엔드와 백엔드를 포함한 전체 프로젝트입니다.

## 📁 폴더 구조

```
company-fullstack/
├── client/   # React + Vite 기반 프론트엔드
└── server/   # 백엔드 (Node.js or Spring 기반)
```

## 🚀 사용법

### 클라이언트 실행

```bash
cd client
npm install
npm run dev
```

### 서버 실행

```bash
cd server
npm install
npm run start
```

## 🔒 환경 변수

`.env` 파일은 로컬에서만 관리하며 Git에는 포함되지 않습니다.
